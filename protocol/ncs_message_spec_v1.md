NCS Message Protocol v1

Purpose:
Define a structured unit of reasoning for inter-agent coordination.

Principles:

Every message targets a versioned decision

Every claim must include justification

Messages must be machine-comparable

Messages must be historically persistent

Core Fields:

protocol_version

message_id

timestamp

agent identity

decision reference

action type

structured arguments

risk analysis

evaluation metrics

consensus signal
